# Examples

See the `examples/` folder in the repository for runnable scripts demonstrating:

- `examples/sync_example.py` — sync retry with `TokenBucket` and `CircuitBreaker`.
- `examples/async_example.py` — async retry example using the async path.

Run them from the repo root:

```bash
python -m examples.sync_example
python -m examples.async_example
```
