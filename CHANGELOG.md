# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-30

### Added
- CircuitBreaker: simple circuit breaker with half-open recovery
- TokenBucket: token-bucket rate limiter
- OpenTelemetry and Prometheus adapter modules (optional integrations)
- Structured logging and lifecycle events for retry attempts
- Integration tests for resilience features
